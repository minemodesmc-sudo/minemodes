## استخدم كملحق

This repository can be added as an **extension** in MakeCode.

* open [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* انقر على **مشروع جديد**
* انقر على ** ملحقات ** ضمن قائمة عجلة التروس
* ابحث عن **https://github.com/minemodesmc-sudo/minemodes** واسترده

## تعديل المشروع

To edit this repository in MakeCode.

* open [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* انقر على ** استيراد ** ثم انقر على ** استيراد عنوان **
* paste **https://github.com/minemodesmc-sudo/minemodes** and click import
